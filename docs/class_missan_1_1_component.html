Detailed Description

template<class T>
class Missan::Component< T >

This uses the "curiously recurring template pattern" (CRTP) so that every class does not have to provide its own implementation of the Clone method: less boiler plate code. Also, provides a convenient and syntactically appealing way of gathering all instances of Component classes. There is a one-to-one correspondence between each subclass T and Component<T>.
