◆ Instantiate()

static GameObject * Missan::GameObject::Instantiate ( GameObject *  original = nullptr)
static

Instantiates a new GameObject calls Start() at end of frame. If an original GameObject is provided, its new instances of all its components are created and attached to the new GameObject. The cloned components will be copies of the original components, but will have their references to gameObject and transform set correctly, but all other fields are simply copied as is.
