Detailed Description

Class for GameObjects that are affected by physics. GameObjects with RigidBodies are automatically simulated by the physics engine.

glm::vec3  point = { 0, 0, 0 },
bool  useLocalSpace = false

Applies force to the RigidBody at point, using world coordinates unless specified otherwise. Force applied anywhere but center of mass (i.e. {0, 0, 0} ) will generate torque. Force, unlike an impulse, is applied continuously once added, e.g. gravity is only added once but affects the RigidBody each frame
